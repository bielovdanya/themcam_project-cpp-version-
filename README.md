themcam_project (cpp) version 1.1

This project implements thermal imaging using an infrared sensor MLX90640 and a TFT LCD 3.5" (ILI9488) display controlled by an ESP32-S3 microcontroller. The sensor reads temperatures from a 32Ã—24 dot matrix in real time, and the display displays a color heat map with temperature values. Minimum and maximum temperature values are also displayed at the bottom. A button allows you to stop/resume image updates for convenient analysis.
